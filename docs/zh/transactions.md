# 交易記錄

PennyWallet 有四種交易類型，每種都針對特定的現實場景設計。

---

## 交易類型

### 支出

從你的帳戶中支付購物或付款的金額。

| 欄位 | 必填 | 說明 |
|------|------|------|
| 帳戶 | 是 | 支付來源帳戶 |
| 分類 | 否 | 例如：餐飲、交通、購物 |
| 備註 | 否 | 自由輸入說明 |
| 金額 | 是 | 正數 |

**對餘額的影響：**
- 現金／銀行帳戶 → 餘額減少
- 信用卡 → 未付款項增加

**範例：** 用現金支付 NT$280 的午餐
→ 帳戶：`現金`，分類：`餐飲`，金額：`280`

---

### 收入

進入帳戶的款項。

| 欄位 | 必填 | 說明 |
|------|------|------|
| 帳戶 | 是 | 收款帳戶 |
| 分類 | 否 | 例如：薪資、獎金、副業 |
| 備註 | 否 | 自由輸入說明 |
| 金額 | 是 | 正數 |

**對餘額的影響：**
- 任何帳戶類型 → 餘額增加

**範例：** 月薪匯入玉山銀行
→ 帳戶：`玉山銀行`，分類：`薪資`，金額：`72000`

---

### 轉帳

在自己的兩個帳戶之間移動資金，無需分類。

| 欄位 | 必填 | 說明 |
|------|------|------|
| 來源帳戶 | 是 | 轉出帳戶 |
| 目標帳戶 | 是 | 轉入帳戶（必須與來源帳戶不同） |
| 備註 | 否 | 例如：`轉至儲蓄帳戶` |
| 金額 | 是 | 正數 |

**對餘額的影響：**
- 來源帳戶 → 餘額減少
- 目標帳戶 → 餘額增加

**範例：** 從 ATM 提領 NT$8,000 現金
→ 來源：`玉山銀行`，目標：`現金`，金額：`8000`

---

### 還款

用銀行或現金帳戶繳清信用卡帳單。

| 欄位 | 必填 | 說明 |
|------|------|------|
| 來源帳戶 | 是 | 必須是現金或銀行（不能是信用卡） |
| 目標帳戶 | 是 | 必須是信用卡 |
| 備註 | 否 | 例如：`繳卡費` |
| 金額 | 是 | 正數 |

**對餘額的影響：**
- 來源帳戶（現金／銀行）→ 餘額減少
- 目標帳戶（信用卡）→ 未付款項減少

**範例：** 從儲蓄帳戶繳清 NT$5,200 信用卡帳單
→ 來源：`玉山銀行`，目標：`Visa 白金卡`，金額：`5200`

> 請參閱 [信用卡流程](./credit-card-workflow) 了解完整的信用卡週期。

## 新增交易

**從財務總覽或交易記錄檢視：** 點擊 **+ 新增交易**

**從指令面板：** 執行 `PennyWallet: Add Transaction`

**從側邊欄圖示：** 點擊氣球圖示 → **+ 新增交易**

**從 iOS 捷徑：** 請參閱 [URI Handler 與 iOS 捷徑](./uri-handler)

---

## 編輯與刪除

開啟**交易記錄**檢視，找到該筆記錄，點擊右側的**編輯（✏）**或**刪除（🗑）**圖示。

編輯支援更改**日期**（包含將交易移至不同月份）、類型、帳戶、分類、備註和金額。

## 預設分類

### 支出
`餐飲` · `交通` · `購物` · `娛樂` · `醫療` · `居家` · `其他`

### 收入
`薪資` · `獎金` · `副業` · `其他`

自訂分類可在 **設定 → PennyWallet → 自訂分類** 中新增。
