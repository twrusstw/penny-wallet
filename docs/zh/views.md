# 介面與檢視

PennyWallet 有三個檢視。**帳本總覽**可從側邊欄圖示開啟；其他兩個檢視可從帳本總覽標題列的按鈕進入，或透過指令面板開啟。

---

## 帳本總覽

主要儀表板。點擊左側面板的 **PennyWallet 圖示**，或從指令面板執行 **PennyWallet: Open Finance Overview** 開啟。

標題列另有三個按鈕：
- **明細** — 切換至收支明細
- **財務趨勢** — 切換至趨勢圖表檢視
- **+ 新增交易** — 開啟交易表單

### 月份導覽

使用 `‹` / `›` 切換月份。未來月份無法選取。

### 摘要指標

| 指標 | 說明 |
|------|------|
| 收入 | 本月記錄的總收入 |
| 支出 | 本月記錄的總支出 |
| 結餘 | 本月收入減去支出 |

### 帳戶餘額

顯示每個使用中帳戶的**即時累計餘額**，從初始餘額開始計算所有交易 — 不限於當月。

信用卡餘額顯示為負數（未付款項）。

底部的**淨資產**為所有現金／銀行餘額之和減去所有信用卡欠款。

### 資產配置圓餅圖

當你有兩個以上現金/銀行帳戶且餘額為正時顯示。呈現流動資產在各帳戶間的分佈。圖例每項顯示帳戶名稱、金額與佔比。

### 分類圓餅圖

若有資料，會顯示兩個圓餅圖：
- **支出分類** — 本月消費的分類佔比
- **收入分類** — 本月收入的分類佔比

圖例每項顯示分類名稱、金額與佔比。將滑鼠移至扇形或圖例項目上可以醒目顯示。

---

## 收支明細

顯示所選月份所有交易的完整清單，包含篩選器和小計。

### 篩選器

- **類型標籤** — 多選：全部 / 支出 / 收入 / 轉帳 / 還款（可同時選多種）
- **分類下拉選單** — 以 checklist 形式列出目前結果中出現的分類，可多選進一步縮小範圍
- **關鍵字搜尋** — 篩選備註包含搜尋文字的交易

### 交易列

每列顯示：日期、類型標籤、分類、備註、帳戶（轉帳顯示來源 → 目標）和金額。

點擊 **✏** 編輯或 **🗑** 刪除。刪除前會出現確認對話框。

### 小計

底部固定顯示目前篩選交易的**支出小計**和**收入小計**。清單可獨立捲動，不影響標題列或小計列。

---

## 財務趨勢

長期財務趨勢檢視。點擊帳本總覽標題列右上角的**財務趨勢**按鈕開啟。

> 財務趨勢沒有專屬的指令面板指令 — 請先開啟帳本總覽（`PennyWallet: Open Finance Overview`），再點擊 **財務趨勢**。

### 時間範圍選擇

選擇 **3 個月**、**6 個月**或 **12 個月**。

### 月度收支長條圖

並排顯示每月收入（綠色向上）和支出（橘色向下）的長條圖。將滑鼠移至欄位上可在提示框中查看精確數字。

### 分類趨勢折線圖

顯示單一分類每月金額的折線圖。透過下拉選單切換分類，圖表即時更新而不重置頁面。圖表下方顯示所選時間範圍的合計金額。

### 淨資產趨勢折線圖

顯示每月末淨資產的折線圖。將滑鼠移至資料點附近可查看數值。缺少資料的月份（無交易）會在折線上形成缺口。

### 帳戶餘額趨勢折線圖

多條折線顯示各現金/銀行帳戶在所選時間範圍內的累計餘額變化。將滑鼠移至圖表上可查看各帳戶數值。需有兩個以上此類帳戶才會顯示。

### 摘要指標

| 指標 | 說明 |
|------|------|
| 月均收入 | 有資料的月份中的平均收入 |
| 月均支出 | 有資料的月份中的平均支出 |
| 淨資產變化 | 該時間範圍內第一個與最後一個資料點之間的淨資產變化 |
